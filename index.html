Portfolio

The Anime Index

This project sends a GET request to an Azure API that I created and allows you to query a number of different animes and returns a list of facts in a clean and user friendly way. The search does exception handling and the CSS is built from the ground up.

Tools and languages used: C#, Blazor Web Assembly, JavaScript and HTML/CSS.

Data Growth Tool

This tool was made as a reaction to Workhuman's overwhelming data growth over the last few years, as a means to track it over months and years. It allows you to import files generated from a script and view them in a graphical way.

It was built to be modular and easy to maintain.

Tools and languages used: JavaScript, AJAX, SQL, UNIX scripting and HTML/CSS.

Happy Apple

This is an online food shop that has a frontend complete with a custom admin backend. You can place orders and write reviews on products. Dedicated profile types for admins, users, delivery drivers and a package tracking system.

Tools and languages used: Python, Django, JavaScript, SQL and HTML/CSS.

Flipper

This is a social media website that's heavily inspired by Twitter and Reddit's layouts. It has a post function along with replies and user profiles. Later iterations included a fully functioning merch store with Stripe integration.

Tools and languages used: Python, Django, JavaScript, SQL and HTML/CSS.
